# Photo & measurement capture checklist

Part 7 is emphatic: **measure first, design second.** Fill in the table below from
your specific chassis before finalizing the `Edge.Cuts` outline — every surviving
1960s stamped-steel chassis differs.

## Chassis / PCB measurement table

| Measurement | Value | Notes |
|-------------|-------|-------|
| Chassis internal length (mm) | | along the long axis |
| Chassis internal width (mm) | | |
| Usable PCB length = length − 15 mm (mm) | | per Part 7 clearance rule |
| Usable PCB width = width − 15 mm (mm) | | |
| Chosen Edge.Cuts size (mm) | | 190×115 if it fits; else 155×90 (errata #9) |
| Vertical clearance PCB→control panel (mm) | | gates main-cap mounting (Part 7) |
| Tallest component height (main filter cap) (mm) | | split to 2×2200µF if tight |
| Original mounting-hole positions | | match 25-5274-2; measure x/y of each |
| Heatsink-bracket location | | LM1875 mounts where germaniums were |

## Photos to take (before / during / after)

**Before teardown**
- [ ] Full chassis, top and bottom
- [ ] Front control panel (all knob positions/labels)
- [ ] Rear panel (jacks, fuse, power entry)
- [ ] Under-chassis overview before removing anything
- [ ] Close-up of original PCB 25-5274-2 (both sides)
- [ ] Transformer T1 wiring + any markings
- [ ] Heatsink bracket + original output devices
- [ ] Footswitch DIN wiring at the chassis

**During**
- [ ] Wire colors/routing as you remove the loom (label both ends)
- [ ] Pot and jack lug wiring before desoldering
- [ ] Reverb pan part-number sticker (confirm 4FB2A1C vs 4AB3C1B)
- [ ] MRB inductor + its DCR reading

**After / build progress**
- [ ] New PCB populated by stage (PSU → preamp → effects → power amp)
- [ ] DC-rail readings at each test point
- [ ] Final install in chassis

Keep filenames descriptive (`01-chassis-bottom.jpg`, `08-T1-secondary.jpg`).
