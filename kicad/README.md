# kicad/

KiCad project files.

- `*.kicad_pro` — project
- `*.kicad_sch` — schematics (8 hierarchical sheets)
- `*.kicad_pcb` — board layout
- `symbols/`    — custom symbol library (VTL5C1, reverb tank, 6-pin DIN)
- `footprints/` — custom footprints (VTL5C1, 1H toroid, 35-pad wiring edge array)
- `gerbers/`    — exported Gerbers (regenerated; see .gitignore)
